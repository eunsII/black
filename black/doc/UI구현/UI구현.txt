UI구현
	
css, js 파일을 따로 분리해서 작성하세요.

js 내의 코드는 jquery 함수를 사용해서 처리해야한다.

아이디는 4~10글자사이로 영문대소문자 숫자를 사용할 수 있다.

비밀번호는 영문대소문자 숫자를 사용해서 4 ~ 10로 만들어져야 한다.
	참고 ] 
		영문대소문자 각 1글자 숫자 1개를 반드시 포함하도록
		정규 표현식을 검사하는 경우 가점 대상...

비밀번호 확인 이벤트는 한글자를 입력하면 체크하는 것으로 하고
다르면 붉은 글씨로 다르다는 메세지가 하단에 표시되어야 하고
같으면 푸른색으로 같다는 표현이 되어야 한다.

메일 아이디는 영문대소문자, 숫자 를 이용해서 4~10 로 한다.
전화번호는 010-1111-1111 의 형식을 사용하기로 한다.

각 성별을 선택하면 해당 성별의 아바타만 보여지도록 하는 경우
가점 대상이 된다.
